# Megaman
# Info
This engine provides the essential tools for creating your own Megaman fangames.

This project has:
- A character controller where you can
    - Walk
    - Jump
    - Extra Jumps
    - Climb
    - Slide
    - Shoot
    - Get damage
    - Die
- Options Menu
- Add custom weapons
- Weapons Menu (In progress)
